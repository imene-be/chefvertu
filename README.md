# Chef Vertu

Site vitrine pour Chef Vertu — une cuisine sans gluten, savoureuse et ancrée dans la science.

## Pages

- **Accueil** — présentation de la démarche, liste des troubles liés au gluten, approche en 3 piliers
- **Pourquoi sans gluten ?** — contenu éducatif basé sur les travaux du Dr David Perlmutter, programme 4 semaines, idées reçues
- **Recettes** — catalogue de recettes filtrables par catégorie, modale de détail
- **Mentions légales** — informations légales, RGPD, hébergement

## Stack

- React 19
- React Router DOM 7
- Vite 7
- Styles inline (CSS-in-JS)
- Déployé sur Vercel

## Lancer le projet

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
